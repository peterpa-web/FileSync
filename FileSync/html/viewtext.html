1 Main Toolbar

1.1 Differences Navigation

Toolbar buttons or menu View:

Prev DiffShow previous difference above
Next DiffShow next difference below

1.2 Undo/Redo

All changes are recorded for Undo/Redo using the menu Edit or the toolbar buttons

1.3 Clipboard Functions

Toolbar buttons or menu Edit allows copy/paste using the system clipboard:

CutCut selected lines
CopyCopy selected lines
PasteInsert lines from clipboard

1.4 Edit Functions

The following toolbar buttons are provided to synchronize file contents (from left to right):

Insert beforeInserts the selected lines before the selection
ReplaceReplaces the selected lines >
Insert afterInserts the selected lines after the selection
DeleteDeletes the selected lines

The arrow direction depends on the selected side. Some buttons may be disabled depending on the local R/W mode.

These functions are also available from the menu Edit.

1.8 String Searching

To search for a string the following toolbar is provided:

  • You may enter a string to be searched and press Enter.
  • A previously entered string may be recalled from the list box by pressing the list button.
  • To continue searching you may press F3 or use the buttons left or right of the entered string.
  • A beep will sound if no item is found for the relevant direction.

4 Additional Functions

4.1 Line Editing

  • Double click to start line editing.
  • Insert or delete characters as usual incl. clipboard support.
  • Cancel editing by pressing ESC or leave the window.
  • To save your changes press Enter.
  • To split the current line press Shift / Enter.
  • Combine the edited line with the following one by pressing Ctrl / Enter.
  • Ctrl double click to quick fix line differences.
  • If applied to a highlighted line difference it will be added to the opposite side.
  • If applied to a common line part the contained differences are removed from the opposite side.
  • If applied to highlighed line number area the whole block of different lines is marked and copied to the opposite side.
